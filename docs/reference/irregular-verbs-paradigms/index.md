# Irregular Verbs - Paradigms

- TOC
{:toc}

***

**Paradigm Verbs**:
- sum, esse, fuī, futūrus - to be
- eō, īre, īvī/iī, ītus - to go
- ferō, ferre, tūlī, lātus - to bear, carry
- volō, velle, voluī - to want, wish
- nōlō, nolle, noluī - not to want
- mālō, malle, maluī - to prefer

***
